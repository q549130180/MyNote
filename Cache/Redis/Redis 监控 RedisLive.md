## 一、简介

Redis Live是一个用来监控redis实例，分析查询语句并且有web界面的监控工具，使用python编写。


## 二、环境准备


- OS:Cent OS 7.0
- Python:2.7.10


## 三、安装依赖

- pip install tornado
- pip install redis
- pip install python-dateutil
- pip install argparse
